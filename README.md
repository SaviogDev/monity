# 💰 Monity

Plataforma de controle financeiro desenvolvida para ajudar pessoas a organizarem seus gastos de forma simples, clara e eficiente.

## 🚀 Propósito

O Monity foi criado com o objetivo de dar clareza financeira para usuários comuns, permitindo que saibam exatamente para onde está indo o seu dinheiro e quanto ainda podem gastar.

## 🧱 Estrutura do projeto

* `frontend/` → Interface do usuário (Angular)
* `backend/` → API e regras de negócio (Node.js)
* `docs/` → Documentação e ideias do projeto

## 🔐 Funcionalidades iniciais

* Cadastro e login de usuários
* Registro de gastos
* Controle de renda mensal
* Cálculo em tempo real do saldo disponível
* Dashboard com resumo financeiro

## 📊 Visão futura

* Controle de cartões
* Alertas inteligentes
* Metas financeiras
* Relatórios detalhados

---

## 🧠 Conceito

Simplicidade + Clareza + Controle

O Monity não é apenas um app financeiro, é uma ferramenta para mudança de comportamento financeiro.
